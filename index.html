CASE STUDY

NASA WEBSITE REDESIGN

Redesign of website of Canadian Space agency, the nation space agency
responsible for advancement in innovation and scientfic development in the feld of Space

Project Overview


CSA is the government agency responsible for space exploration and aeronautics in Canada. Users navigate the site to find valuable information such as new articles, latest on space and educational resources available on this site.
User Actions
  • Read about Canadian Northern lights Activity
  • About Astronauts
  • About international space station – Science, Technology robotics
  • Educational resources for young people
  • Careers
  • CSA’s latest and important operations
User Background
  • General Public
  • Student
  • Teacher
  • Researcher
  • Aerospace Industry

Goal

The main goal of this project was to simplify navigation.

Responsibilities

User research
Visual Design

Tools

Figma
Miro

Project Details

Overview

After conducting preliminary interviews on the usability of the original website the conclusion was that the user were getting overwhelmed with the large amount of information on the website. The task there involved identifying areas of the site currently cause frustration or confusion and redesigning to create a great user experience

Duration

4 weeks

Scope

User research, definition, ideation, wireframe, prototype and user testing and improving the UI elements of the website

Key Focus Areas

  • Simplify navigation
  • Declutter
  • Arrange important information hierarchically
  • Present large amount of data without overwhelming the user
  • Enhance User Experience

Define and Ideate

Problem Definition

CSA’s website contains a lot of valuable information, but it is not well organized. It would be particularly beneficially for information seekers & researcher to easily access the resources available on this site. However, confusing categories, visually busy layouts and a lack of hierarchy make navigating asc-csa.gc.ca too time consuming. With Information seekers in mind, we created a more organized and intuitive responsive web design.
Card Sorting

The users were asked to card sort. The groupings were used as the basis for a new site map. With information architecture in place, the redesigning the navigation began.

Information Architecture

User tests revealed that information clutter was an issue on the website and the right styling was imperative to project important information with clarity.

Style tile

The modern and simple typefaces were combined with a palette that matches the deep colours of the space combined with dynamic images. The interface has been transformed from the original white background, to cool colors, that matches the galaxy
